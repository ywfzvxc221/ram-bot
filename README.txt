
كيفية تشغيل البوت:

1. تثبيت المتطلبات:
   pip install -r requirements.txt

2. تجهيز متغيرات البيئة (Env Vars) داخل الاستضافة:
   - BOT_TOKEN : توكن البوت
   - CHANNEL_USERNAME : اسم القناة مع @
   - TASK_REWARD : قيمة مكافأة تنفيذ المهمة
   - REF_REWARD : قيمة مكافأة الإحالة
   - REF_PERCENT : نسبة أرباح الإحالة
   - ADMIN_ID : آي دي الأدمن

3. تشغيل البوت:
   python bot.py

جاهز للاستخدام!
